APIs used
  • NOAA CO-OPS Tides & Currents — /predictions hourly tide curve & hi/lo markers (station 9449424 Cherry Point, datum MLLW)
  • NWS Weather API — /points → forecastHourly for hourly temperature, precipitation probability, and wind
  • NWS Weather API — /points → forecastGridData for gridpoint skyCover (cloud cover %)
  • NWS Products — /products/types/CWF/locations/SEW & /products/{id} for Coastal Waters Forecast (zone PZZ133)
  • Sunrise-Sunset — daily /json for sunrise & sunset used for night-time shading
